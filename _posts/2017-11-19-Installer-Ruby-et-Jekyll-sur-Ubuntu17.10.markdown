# Installer Ruby et Jekill 

```
sudo apt-get update
sudo apt-get install ruby ruby-dev ruby-all-dev make gcc
sudo gem install jekyll bundler
```

# Desintaller Ruby et Jekyll

```BASH
sudo apt-get purge ruby
```
